Maintenance Notes

When to update the canonical graph and how to regenerate this explorer.

source of truth: docs/architecture.graph.ts
Do not edit this generated file directly. Run pnpm graph:render after changing the graph. Diagrams load Mermaid from a CDN in this v1 explorer, so rendering needs network access.
  • Add, remove, rename, or materially change a tracked source file.
  • Change a runtime boundary, API route, provider seam, deployment shape, or generated graph tooling.
  • Change framework/data-client contracts, durable entities, tool execution, tenancy, policy, storage, or secret-custody assumptions.
  • Promote a target or planned concept into current implementation.

After graph updates, run pnpm graph:render and pnpm graph:check.
